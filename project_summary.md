# CNE Independent Project 1 - Linux Administration
## Overview
This project is a comprehensive Linux administration challenge designed to demonstrate my ability to independently manage a web environment from initial setup through deployment and troubleshooting. 

I will organize a secure Linux workspace by configuring users, groups, and file permissions, then develop an original shell script to automate administrative tasks. 

The core objective involves installing and configuring NGINX to serve a static website, implementing a specific customization to reflect my work. 

Finally, I must document a structured troubleshooting process for at least 3 distinct issues, proving my capability to diagnose and resolve problems related to services, permissions, and network connectivity in a professional manner.
