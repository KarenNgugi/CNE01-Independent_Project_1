# CNE Independent Project 1 - Linux Administration
## About this project
The objective of this project is to demonstrate practical Linux administration, shell scripting, Git, and automation skills by completing a real-world task independently.

## Objectives
- Navigate and administer a Linux system
- Work comfortably with the command line
- Use Git for version control
- Write reusable Bash scripts
- Automate repetitive tasks
- Troubleshoot common Linux issues
- Document technical work clearly

## Technologies Used
- Linux
- Bash
- Git
- Nginx

## Project Structure
The project will be structured as such:
* **files/** : all files necessary for the project, including:
    * **configurations/** : all files related to configuration 
    * **scripts/** : all files related to scripting
    * **web_files/** : all files for web deployment
* **report/** : the full project report will be located here
* **screenshots/** : all screenshots for the relevant project milestones
    * **milestone_1/** : all screenshots related to Milestone 1: Linux System Setup and Administration Basics
    * **milestone_2/** : all screenshots related to Milestone 2: Shell Scripting and Automation
    * **milestone_3/** : all screenshots related to Milestone 3: NGINX Static Website Deployment
    * **milestone_4/** : all screenshots related to Milestone 4: Troubleshooting & Operations Validation

## Lessons Learned
Through this project I gained practical experience in:
- Linux system administration
- Bash scripting
- Git workflows
- Technical documentation
- Troubleshooting

The project also reinforced the importance of planning, testing, and documenting every stage of implementation.
